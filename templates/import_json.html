Nhập dữ liệu từ file JSON

Chỉ chấp nhận file có định dạng .json
Lưu ý:
  • File JSON phải được xuất từ hệ thống này
  • Việc nhập sẽ tạo một profile mới với tất cả dữ liệu từ file
  • Profile mới sẽ được đặt làm profile hiện tại
  • Dữ liệu hiện tại sẽ không bị ảnh hưởng
Quay lại
Hướng dẫn sử dụng
Cách tạo file backup JSON:
  1. Chọn profile cần backup
  2. Click nút "Xuất JSON" trên trang chủ
  3. File JSON sẽ được tải về máy tính
Cách khôi phục dữ liệu:
  1. Chọn file JSON đã backup trước đó
  2. Click "Nhập dữ liệu"
  3. Hệ thống sẽ tạo profile mới với tất cả dữ liệu từ file
